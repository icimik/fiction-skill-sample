# 世界观

## 理论锚点

1. **托尔金"第二世界"与"次级信仰"**：读者对虚构世界的信任不是"悬置怀疑"，而是当内部逻辑绝对严密时自发产生的真实信任感。哪怕是奇幻设定，也要遵循能量守恒、经济学规律、人性逻辑。

2. **桑德森第一定律**：解决剧情矛盾的核心不在于赋予角色逆天能力，而在于给能力绑定明确的限制、代价与弱点。力量体系三要素必写：限制（使用条件）、代价（消耗/反噬）、弱点（致命短板）。

3. **海明威冰山理论**：作者搭建 100% 完整的世界框架，正文仅释放 1/8 服务于当前剧情的内容。开篇仅讲清生存规则、核心冲突、基础力量；深层历史、隐秘势力、世界真相埋入伏笔随主线揭开。

## 可执行条款

1. 世界观必须覆盖：物理环境、社会文化、规则体系、背史、验证场景。
2. 能力体系必须写明：限制、代价、弱点，不得留空。
3. 命名与映射必须先于正文：机构、地名、平台、组织名称统一。
4. 设定必须有信息释放计划：开篇只放必要规则，深层设定分阶段释放。
5. 每个核心设定至少给出 1 个可被剧情验证的场景。

## 脚手架

1. 搭基础框架（冰山水面之上）：定世界背景、势力阵营、稀缺资源三大核心。
2. 定规则与代价（防崩核心）：搭建力量体系，绑定限制、代价、弱点，标注不可破铁律。
3. 精细化填充：补充世界人文、生活细节、阶层规则，让虚构世界有温度。
4. 埋深层伏笔（冰山水面之下）：设置 2-3 个隐藏设定，支撑长篇续航与剧情反转。
5. 冰山修剪：删除/隐藏暂不服务剧情的设定，精简开篇。

## 产出要求

- `世界观设定.md`
- `规则清单与验证表.md`
- `信息释放计划.md`

## 检查项

- [ ] 规则无冲突（前后文一致）。
- [ ] 无"临时改规则"解决剧情问题。
- [ ] 核心设定能支持至少 3 个剧情单元。
- [ ] 世界细节服务情节，不是纯展示。
- [ ] 架空映射表完整并可复用。
- [ ] 力量体系有明确限制、代价、弱点，无无限开挂。
- [ ] 90% 深层设定藏于水下，开篇无冗长设定堆砌。

## 避坑清单

- 开篇绝不堆设定，通过剧情、对话逐步释放信息。
- 不随意打破设定铁律，哪怕主角高光也不越界。
- 不用生僻晦涩术语，降低阅读门槛。
- 不做悬浮设定，贴合人性与基础逻辑。

## 失败判定

- 出现无代价开挂。
- 出现设定互相矛盾且未解释。
- 开篇信息倾倒导致主线停滞。

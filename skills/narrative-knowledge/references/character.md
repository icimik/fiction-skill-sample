# 角色

> ⚠️ **预览版** — 当前内容为精选预览，获取完整版请[联系我们](https://github.com/icimik/fiction-skill-sample/issues/new?labels=Request+for+Quotation)。

## 理论锚点

1. **荣格"面具与阴影" + 创伤与谎言法则**：立体角色的核心是"外在面具（自我保护的谎言） + 内在阴影（过往创伤） + 成长弧光（直面真相）"。创伤是行为动机源头，谎言是性格外壳，弧光是打破谎言、直面阴影的蜕变过程。弧光遵循"固守谎言 → 冲击破碎 → 直面创伤 → 接纳真相"四步走。

2. **格雷马斯动体模型**：角色关系不是静态网络，而是功能性动力学模型。六大角色功能相互牵制：主体（渴求目标的主角）、客体（核心目标）、发送者（行动诱因）、接受者（受益人）、辅助者（帮手）、反对者（反派/障碍）。

3. **苏格拉底式对话交锋**：角色对话不是信息传递，而是心理博弈、价值观碰撞、痛点试探。通过对话戳破对方面具、触碰内心阴影，既推动剧情，又强化角色立体感。

## 可执行条款

1. 核心角色必须包含：身份、目标、动机、缺陷、创伤、弧光。
2. 主角必须区分"想要"与"需要"，并在剧情中产生张力。
3. 反派必须具备自洽立场，不得仅以"邪恶"驱动。
4. 关系网络必须可视化：盟友、对手、利益冲突、依赖关系。
5. 每个关键角色必须有至少 1 个高光行为节点。

## 脚手架

1. 塑核心主角：敲定外表标签、性格反差、天赋短板，深挖创伤、谎言、弧光，设定言行铁律。
2. 造核心配角：选定 3 个核心配角（战力/智囊/气氛担当），赋予专属动机、高光、人设。
3. 塑反套路反派：赋予反派创伤、立场、自洽理念。
4. 搭关系网：梳理角色羁绊/对立/博弈关系，构建动体模型，设计苏格拉底式交锋点。
5. 定人设红线：明确角色言行底线。

## 产出要求

- `角色卡.md`
- `关系矩阵.md`
- `角色弧线表.md`

## 检查项

- [ ] 关键行为能追溯到角色动机。
- [ ] 不存在明显 OOC 行为。
- [ ] 配角不是工具人（有独立目标）。
- [ ] 关系变化有触发事件支持。
- [ ] 角色成长不是口头宣告而是行为变化。
- [ ] 主角有深层创伤与清晰动机，行为逻辑自洽。
- [ ] 反派动机合理，具备共情点与理念支撑。
- [ ] 关系网有张力、有博弈，一人变故牵动全员。

## 避坑清单

- 不堆砌冗余配角，仅保留核心角色。
- 不强行煽情，角色情感靠剧情铺垫。
- 不突破人设铁律，哪怕剧情需要也不 OOC。
- 配角不全程衬托主角，赋予专属戏份与高光。
- 忌角色关系单一，打造羁绊、对立、亦敌亦友的张力感。

## 失败判定

- 角色行为服务作者意图而非人设逻辑。
- 角色关系只靠设定说明，无剧情体现。
- 反派动机空洞导致冲突失真。
